  DIMENSION LOCK :: DEATHLY STORIES — PORTABLE EDITION

HOW TO PLAY
-----------
 - WINDOWS:  Double-click PLAY-Windows.bat   (or PLAY.html)
 - MAC:      Double-click PLAY-macOS.command (or PLAY.html)
 - LINUX:    Run        ./PLAY-Linux.sh      (or PLAY.html)

Anywhere a recent browser is installed (Chrome, Edge, Firefox,
Safari), this bundle runs without an internet connection.

WHAT'S INSIDE
-------------
 - Main Menu with Maytradalis lore intro and link to the comic.
 - Astrolabe Terminal: full 3D 199-strata creation explorer.
 - Reality Breach Defense: arcade mini-game.
 - Procedural ambient Dimensionlock dirge (auto-plays on tap).
 - Lovecraftian / demonic / angelic procedural sub-locations.

USING YOUR OWN MUSIC
--------------------
Drop your MP3 file at:
   data/api/static/dimensionlock_theme.mp3
The menu will auto-detect and play it instead of the dirge.

COMMUNITY FEATURES
------------------
Community Lore Contributions and Community Universes are OFFLINE
in this portable bundle. To enable them, deploy the FastAPI
backend separately and edit data/api/static/config.js to point
at it (apiBase: "https://your-server.com").
